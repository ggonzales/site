Web Developer
Creates and delivers effective and efficient multi-platform web applications; as well as supervise a team to serve the same purpose.
Duties and Responsibilities

  1. Complete applications development by coordinating requirements, schedules, and activities; organizing and contributing to team meetings; troubleshooting development and production problems across different platforms.
  2. Develop, test, debug and document efficient and elegant applications consistent with the requirements specified for the business.
  3. Organize and participate in all of the development process: creating high quality code, ensuring less code deficiency, and engaging effectively in code reviews.
  4. Contribute ideas and/or facilitate trainings to share knowledge for the benefit of the project and the whole team.
  5. Supervise a team of Junior Web Developers.
  6. Updates job knowledge by researching new technologies and tools; participating in educational opportunities; reading professional publications.
Qualifications

Note. All the requirements are absolutes unless listed as desirable

  • Education
    - Bachelor's Degree in Computer Science, Mathematics, Information Systems, Software Engineering or equivalent.

  • General Requirements
    - Strong foundation in web development using HTML5, JavaScript, jQuery and CSS3, with knowledge in animation using HTML5 and jQuery UI.
    - Strong understanding of OO concepts in JavaScript context and knowledge in DOM Event Models.
    - Passionate about development and eager to learn new technologies.
    - Experience with the development life cycle starting from coding, unit testing and code reviews.
    - Able to communicate, share knowledge and reach out to peers to solve problems and/or create effective algorithms.
    - Able to supervise a team of junior developers.
    - Demonstrate commitment to quality.
    - Action oriented and results driven.
    - Able to work independently, on several concurrent tasks with changing priorities.
    - Source Control management.
    - Ability to extract requirements from high-level business rules and user explanations.
    - Requirements analysis, use and test case development, workflow technologies and system application design


  • Experience
    - 3 to 4 years of proven web development experience using HTML, JavaScript, CSS and other web development tools and libraries.
